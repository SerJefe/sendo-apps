### Keyboard Shortcuts
- **Cmd/Ctrl + O**: Quick switcher (find any note)
- **Cmd/Ctrl + E**: Edit/Preview toggle
- **Cmd/Ctrl + P**: Command palette (all functions)
- **Cmd/Ctrl + G**: Open graph view
- **[[**: Link to any note (auto-suggest)

## 🆘 Troubleshooting

### Common Issues
**Templates not working**: Check Settings → Templates → Template folder path
**Daily notes broken**: Settings → Daily Notes → Date format + template path
**Slow performance**: Too many plugins OR large attachments
**Sync conflicts**: Close Obsidian on other devices before major edits
**Missing links**: Use [[ syntax, not markdown links
