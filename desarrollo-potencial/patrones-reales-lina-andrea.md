# Patrones Reales: Lina y Andrea — Datos Concretos de Sebastián

## Lina: Potencial comercial sin canal

### Perfil observado por Sebastián:
- **"No hemos montado nada"** → Potencial sin estructura
- **"Ve esto como su oportunidad de crecer, asociarse, desarrollar"** → Motivación intrínseca alta
- **"Gestiona mucho y tiene potencial comercial"** → Capacidades existentes no aprovechadas
- **"Le das algo y empieza a ver cómo hacerlo"** → Alta iniciativa una vez activada
- **"Le gusta producir, no quedarse quieta"** → Energía productiva natural

### Patrón de desarrollo:
**PATROCINIO = Estructurar energía existente**
- Tiene el motor (ganas de producir)
- Necesita la dirección (hacia dónde canalizar)
- Patrocinio = darle oportunidades para que ella vea cómo desarrollarlas
- No necesita motivación, necesita marco

## Andrea: Lógica fuerte con aspectos a expandir

### Perfil observado por Sebastián:
- **"Es leal"** → Factor confianza fundamental
- **"Quiero que sea la de más confianza que maneje la empresa"** → Objetivo de desarrollo específico
- **"Entienda patrones"** → Capacidad analítica a desarrollar
- **"Muy seria a veces, chistes con sátira"** → Personalidad directa/dura
- **"Fuerte con otros, como si no le importaran"** → Aspecto interpersonal a suavizar
- **"Lógica muy buena"** → Fortaleza base confirmada
- **"Hay que enseñarle lo que no es lógico"** → Expandir más allá de lógica pura
- **"Curiosa, mejora cosas"** → Mentalidad de crecimiento activa

### Patrón de desarrollo:
**RESPONSABILIDAD = Expandir capacidad existente**
- Tiene la base (lógica, lealtad, curiosidad)
- Necesita amplitud (patrones, aspectos no lógicos, suavidad interpersonal)
- Responsabilidad = ponerla en situaciones que la obliguen a crecer
- No necesita estructura, necesita desafío

## Marcos conceptuales refinados

### 1. "Potencial vs Capacidad"
**Lina (Potencial sin canal):**
- Motor fuerte, dirección difusa
- Patrocinio = crear el canal para su energía
- "Oportunidad" es su palabra clave

**Andrea (Capacidad sin amplitud):**
- Dirección clara, motor lógico
- Responsabilidad = forzar expansión beyond lógica
- "Confianza" es su palabra clave

### 2. "Tipo de crecimiento"
**Lina = Crecimiento por exploración:**
- Dale un territorio y ella lo explora
- "Empieza a ver cómo hacerlo" → mentalidad discovery
- Necesita espacio seguro para experimentar

**Andrea = Crecimiento por profundización:**
- Dale un problema y ella lo resuelve
- "Mejora cosas" → mentalidad optimization
- Necesita desafíos que la obliguen a salir de lógica pura

### 3. "Obstáculos diferentes"
**Lina:**
- Obstáculo: dispersión de energía
- Solución: patrocinio que enfoque sin limitar

**Andrea:**
- Obstáculo: rigidez lógica/interpersonal
- Solución: responsabilidades que requieran flexibilidad

## Aproximaciones conceptuales emergentes

### "Desarrollo por Activación vs Desarrollo por Expansión"

**Activación (Lina):**
- Potencial dormido que necesita despertar
- Patrocinio = crear condiciones para que se active
- Una vez activada, genera momentum propio

**Expansión (Andrea):**
- Capacidad activa que necesita crecer
- Responsabilidad = empujarla beyond zona de confort
- Crecimiento continuo dentro de marco existente

### "Motivación intrínseca diferente"

**Lina = Motivación por oportunidad:**
- "Esto es MI chance"
- Ve el futuro posible y se energiza
- Necesita ver el panorama grande

**Andrea = Motivación por competencia:**
- "Puedo hacer esto bien"
- Se activa con problemas que resolver
- Necesita ver el desafío específico
