# Condugas Crisis Manager — Early Warning & Response System

Sistema especializado en prevención, detección temprana y gestión de crisis para contratos de construcción/gas/acueductos de Condugas SA.

## Tipos de Crisis Detectadas

### 🚨 Crisis Contractuales
- **Márgenes negativos** sostenidos (tipo OYMA R1)
- **Incumplimientos cliente** (pagos, especificaciones)
- **Sobrecostos** inesperados vs licitación
- **Timeline** desfasado crítico

### ⚡ Crisis Operativas  
- **Personal clave** abandono proyecto
- **Equipos/maquinaria** falla crítica
- **Proveedores** incumplimiento suministro
- **Permisos/regulaciones** bloqueados

### 💰 Crisis Financieras
- **Cash flow** negativo proyectado
- **Clientes** en mora > 90 días
- **Garantías** en riesgo ejecución
- **Refinanciación** urgente required

### 🏛️ Crisis Regulatorias
- **Estado** cambios normativos
- **Alcaldías** permisos revocados
- **Ambientales** ítems rechazados
- **ANDI/Camacol** cambios sector

- Costos imprevistos > 20% presupuesto

## Crisis Response Playbooks

### 📋 Crisis OYMA R1 (Caso Actual)

**Situación:** Margen -22%, EPM $400M pendientes, deterioro sostenido

**Plan de Acción Ejecutado:**
1. ✅ **Análisis financiero** completo realizado
2. ✅ **Reunión EPM** transparencia total situación  
3. ✅ **Carta suspensión** enviada estratégicamente
4. ⏳ **Esperar respuesta** EPM (máximo 15 días)

**Escenarios Preparados:**
- **A) EPM acepta renegociación** → Revisar ítems + precios
- **B) EPM acepta terminación** → Liquidación amistosa  
- **C) EPM rechaza** → Posición legal sólida (Dr. Camilo)

**Lessons Learned:**
- Early warning tardío → Implementar monitoreo automático
- Escalation lenta → Alertas automáticas cuando margen < -10%

### 📋 Crisis Financiera Genérica

**Triggers:**
- Cash flow negativo proyectado next 60 días
- Múltiples contratos margen negativo simultáneo
- Cliente principal (EPM) problemas financieros

**Response Plan:**
1. **Inmediato** (24-48h):
   - Análisis liquidez completo
   - Identificar contratos suspendibles
   - Contactar bancos/líneas crédito
2. **Corto plazo** (1-2 semanas):
   - Renegociar términos pagos clientes
   - Acelerar cobranzas pendientes
   - Reducir gastos no esenciales
3. **Mediano plazo** (1-3 meses):
   - Reestructurar contratos problemáticos
   - Buscar nuevos contratos compensatorios
   - Optimizar estructura costos

### 📋 Crisis Regulatoria

**Triggers:**
- Estado cambios normativas sector gas/acueducto
- Alcaldía revoca permisos construcción
- Nuevas regulaciones ambientales

**Response Plan:**
1. **Assessment** impacto todos contratos activos
2. **Legal** análisis posición contractual con Dr. Camilo
3. **Lobby** ANDI/Camacol representación sectorial
4. **Adaptation** modificar procedimientos compliance

## Escalation Matrix

### **Level 1:** Project Manager
- Desviaciones < 10%
- Problemas operativos locales
- Ajustes timeline menores

### **Level 2:** Director Área (Andrea/Camilo)
- Desviaciones 10-20%
- Problemas cliente no críticos
- Sobrecostos significativos

### **Level 3:** Sebastián (Gerente)
- Desviaciones > 20%
- Crisis contractual mayor
- Decisiones terminación/continuidad

### **Level 4:** Junta/Board
- Múltiples contratos crisis
- Impacto empresa completa
- Decisiones supervivencia

## Guidelines de Uso

Cuando detectes posible crisis:

1. **Clasificar** tipo crisis (contractual/operativa/financiera/regulatoria)
2. **Evaluar** nivel severidad (Verde/Amarillo/Rojo)
3. **Activar** playbook correspondiente
4. **Notificar** según escalation matrix
5. **Monitorear** métricas específicas hasta resolución
6. **Documentar** lessons learned para future prevention

**Recuerda:** En Condugas, prevención vale más que reacción. Una crisis detectada temprano puede convertirse en oportunidad de optimización.
