# Alletagia Interventions — Framework Intervención Jefatura

Sistema de diagnóstico e intervención basado en modelo **Ser Siendo** y framework **Alletagia** para desarrollo del potencial humano en contexto empresarial.

## Fundamentos Ser Siendo

### 🌟 Principio Central
**"El humano de hoy no es el de mañana"** → Desarrollo continuo del potencial máximo individual.

### ⚡ Coherencia = Emoción + Pensamiento + Acción Alineados
- **Emoción:** Estado interno genuino
- **Pensamiento:** Análisis situacional claro  
- **Acción:** Comportamiento coherente con emoción + pensamiento

### 🎯 Tres Decisiones Humanas:
1. **Sobrevivir** → Respuestas básicas supervivencia
2. **Recuperar Control** → Gestión situaciones complejas
3. **Desarrollarse** → Crecimiento potencial máximo

### 🔥 Tres Motivadores:
1. **Identidad** → Quién soy/quién quiero ser
2. **Autonomía** → Capacidad decisión/acción propia
3. **Disfrutar** → Satisfacción genuina con proceso/resultado

## Framework Alletagia (Intervención Jefatura)

### 📊 Diagnóstico Energético

#### **Energía Expansiva** (Tipo Lina):
- **Características:**
  - Potencial alto, estructura baja
  - Creatividad, ideas, visión amplia
  - Necesita patrocinio, no control
  - Energía contagiosa, inspiracional
  
- **Intervención Requerida: PATROCINIO**
  - Dar espacio exploración
  - Proteger de micromanagement
  - Facilitar recursos sin estructura rígida
  - Celebrar experimentos/aprendizajes

#### **Energía Intensiva** (Tipo Andrea):
- **Características:**
  - Capacidad alta, necesita amplitud
  - Ejecución, resultados, profundidad
  - Necesita responsabilidad clara
  - Energía enfocada, sistemática

- **Intervención Requerida: RESPONSABILIDAD**
  - Asignar proyectos retadores
  - Dar autonomía decisional
  - Establecer métricas claras
  - Reconocer logros específicos

### 🎯 Matriz Intervención

```
POTENCIAL vs ESTRUCTURA

Alto Potencial + Baja Estructura = PATROCINIO
(Caso Lina: energía expansiva)

Alta Capacidad + Necesita Amplitud = RESPONSABILIDAD  
(Caso Andrea: energía intensiva)

Bajo Potencial + Alta Estructura = COACHING
(Desarrollar competencias específicas)

Baja Capacidad + Baja Estructura = REUBICACIÓN
(Buscar mejor fit persona-rol)
```

## Principios Intervención

### 🚫 ANTI-PATTERNS (No hacer):
- **Nunca** usar "subordinado" → usar "colaborador" 
- **Nunca** usar "liderazgo" → usar "jefatura"
- **No** micromanagement en energías expansivas
- **No** dar solo estructura sin autonomía a intensivos
- **No** intervenciones genéricas → personalizar según tipo

### ✅ PATTERNS EFECTIVOS:
- **Sempre** partir del potencial, no las debilidades
- **Siempre** alinear intervención con tipo energía natural
- **Sempre** dar autonomía proporcional a capacidad demostrada
- **Siempre** celebrar coherencia emoción-pensamiento-acción

## Proceso Diagnóstico

### 🔍 Evaluación Inicial:
1. **Observar** comportamientos naturales sin presión
2. **Identificar** fuente motivación (identidad/autonomía/disfrute)
3. **Clasificar** tipo energía (expansiva vs intensiva)
4. **Evaluar** coherencia emoción-pensamiento-acción actual

### 📋 Preguntas Diagnósticas:
- "¿En qué momentos te sientes más auténtico/coherente?"
- "¿Qué necesitas para dar lo mejor de ti?"
- "¿Cuándo sientes que pierdes energía vs cuándo la ganas?"
- "¿Prefieres explorar posibilidades o profundizar en lo conocido?"

### 🎯 Diseño Intervención:
1. **Seleccionar** approach (patrocinio/responsabilidad/coaching/reubicación)
2. **Personalizar** según contexto específico persona
3. **Establecer** indicadores éxito observables
4. **Planear** seguimiento y ajustes

### 📊 Monitoreo Resultados:
- **Comportamental:** ¿Cambios observables energía/motivación?
- **Resultados:** ¿Mejora performance en rol actual?
- **Coherencia:** ¿Mayor alineación emoción-pensamiento-acción?
- **Crecimiento:** ¿Busca nuevos desafíos/responsabilidades?

- Transiciones rol/responsabilidades
- Intervenciones post-crisis organizacional
