Paso 5: Procedimiento de Escalamiento

¿Qué hacer si la conciliación automática falla?
  1. Verificar conexión API bancaria (Panel → Conexiones → Estado)
  2. Revisar reglas de conciliación (¿coincidencias <80%?)
  3. Validar formato extractos (¿cambió el banco el formato?)
  4. Contactar soporte Odoo si persiste >2 días
  5. Revertir a manual temporal solo si es crítico
Nunca hacer conciliación manual sin antes intentar arreglar la automática. El objetivo es 100% automatización, 0% manual.
