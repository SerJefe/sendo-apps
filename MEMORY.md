## GLOSARIO COMPLETO MODELO SER.SIENDO — MEMORIZACIÓN TOTAL

### ORIGEN FILOSÓFICO
- **Integra**: Heidegger (Dasein), Damasio (neurociencia), Kahneman (psicología cognitiva), teoría sistemas vivos
- **Principio base**: "Solo se es siendo" — el ser no es estático, es proceso continuo
- **El ser emerge del encuentro/interacción, NO del concepto**

### CONCEPTOS NUCLEARES
- **SER.SIENDO** = SER (esencia, raíz, valores, historia, genética interna) + SIENDO (proceso, movimiento, habitar mundo hoy, verbo encarnado)
- **Coherencia** = emoción + pensamiento + acción ALINEADOS (contextual, temporal, NO absoluta)
- **"Coherencia disponible"** = máxima coherencia posible dado el contexto actual
- **"El humano de hoy no es el de mañana"** = principio transformación continua

### TRES DECISIONES HUMANAS (orden obligatorio)
1. **Sobrevivir** — respuesta instintiva, sistema nervioso simpático, lucha/huida
2. **Recuperar el control** — búsqueda rutinas, estabilidad, sistema 2 Kahneman  
3. **Desarrollarse** — crecimiento, fluir, cuando hay seguridad suficiente

### TRES MOTIVADORES
1. **Identidad** — quién soy, reconocimiento, sentido pertenencia
2. **Autonomía** — capacidad decidir, control sobre entorno
3. **Disfrutar** — placer, satisfacción, eudaimonía pragmática

### ALETEAGOGÍA (ΑληθΑγωγία) — PALABRA CREADA POR SEBASTIÁN
- **Etimología**: aletheia (ἀλήθεια) = "verdad desvelada" + agōgós (ἀγωγός) = "el que guía"
- **Definición**: conducción activa hacia la verdad desvelada
- **Práctica**: facilitar retirando velos (ruido personal, prejuicios) para que verdad del otro emerja
- **Estado**: sin mi ruido, sin mi juicio, con apertura para servir descubrimiento del otro
- **Verbo**: "aleteaguiar" — ponerse en modo aleteagogía

### LETAGOGÍA
- **Variante simplificada** de Aleteagogía para contexto práctico
- **Definición**: desarrollo potencial humano a través conversaciones

### JEFATURA (NUNCA "liderazgo")
- **Sebastián RECHAZA palabra "liderazgo"** — vacía y carismática
- **JEFATURA** = responsabilidad real quien toma decisiones que impactan sistema
- **El jefe = sistema nervioso central de la organización**
- **Tres tipos de jefe**:

  2. **El Técnico**: decide desde conocimiento parcial, desconectado del todo
